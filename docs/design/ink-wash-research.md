# 水墨中国风 · 设计调研与应用方案

> 目标：把当前功能完整的 app，从"通用浅色 UI"重构为**水墨中国风**视觉语言——服务于"读中医古籍"的核心场景，让界面本身契合内容气质（古意、克制、留白、气韵）。
>
> 本文是设计**方向与语言**（调研产出），不是逐像素稿。确认方向后按 §10 路线分阶段实施。

---

## 1. 设计理念（美学内核）

水墨中国风不是"贴几个祥云纹样"，而是一套**取舍观**。核心四条，所有视觉决策都回扣到这里：

| 原则 | 含义 | 在本 app 的体现 |
|---|---|---|
| **留白（计白当黑）** | 空白即画，疏可走马 | 大间距、不填满；阅读区四周大量呼吸；卡片少边框、靠空间分隔 |
| **气韵生动** | 有呼吸感、非死板网格 | 微动效（墨晕渐显、卷轴展开）、不规则水墨边缘、手写感字体 |
| **虚实相生** | 实（墨/字）与虚（白/淡）互衬 | 原文（实，浓墨衬线）↔ 解读（虚，淡墨/淡彩）；主次靠浓淡而非加粗 |
| **以少胜多** | 一点朱砂胜过满屏彩色 | 全局基本只黑白灰 + **一点朱砂**（印章/强调），淡彩仅用于点缀 |

> 反模式（要避免）：满屏祥云/回纹底纹、霓虹渐变、阴影厚重卡片网格、emoji 图标、毛笔字字体滥用当装饰。这些是"伪国风"，显廉价。

## 2. 色彩系统

### 2.1 浅色（宣纸 / 默认）

以**宣纸底 + 墨色阶**为骨，**朱砂**为唯一强调，**花青/赭石**极少量点缀（浅绛淡彩）。

```
宣纸底（背景）
  paper        #F4ECD8   主背景（牙黄宣纸）
  paper-warm   #EDE3CC   次背景/卡片底
  paper-cool   #EFE9DA   冷一点的白（解读栏区分）

墨色阶（文字/线条，由浓到淡）
  ink-jiao     #1C1A17   焦墨（标题/最重）— 近黑带暖
  ink-nong     #2B2620   浓墨（正文）
  ink-zhong    #4A4138   重墨（次级文字）
  ink-dan      #6E6356   淡墨（元信息/placeholder）
  ink-qing     #9C9082   清墨（最淡/分隔线）

朱砂（唯一强调色）
  cinnabar     #B23A2E   印章 / 主按钮 / 当前选中 / 关键数字
  cinnabar-soft #C76B5C  朱膘（hover/次级强调）

点缀（淡彩，极少量）
  huaqing      #3E5C6E   花青（链接/信息态）
  zheshi       #8C6A4F   赭石（标签/暖点缀）
  qianjiang    #E4D6BE   浅绛淡染（解读栏背景微暖）
```

### 2.2 深色（夜墨 / 暗色模式）

不是简单反色，而是**浓墨为底、宣纸为字**，模拟夜间书房。

```
  bg           #14110E   浓墨底（带暖，非纯黑）
  surface      #1E1A16   焦墨面（卡片）
  fg           #E8DEC9   宣纸字
  muted-fg     #A89B89   淡墨字
  cinnabar     #C84A3A   朱砂（深底上提亮一档）
  border       rgba(232,222,201,0.12)
```

### 2.3 取舍

- **禁纯黑纯白**（`#000`/`#fff`）——水墨没有纯黑纯白，带暖灰更古。
- 全局饱和度极低（除朱砂），墨阶靠**明度**分层而非色相。
- 当前主题的 `#5C4033 / #EDE4D5 / #A67C5D` 已接近，需补**墨色阶梯度**（5 档灰）+ 朱砂强调。

## 3. 字体

```
--font-title:  "LXGW WenKai", "霞鹜文楷", "STKaiti", "KaiTi", serif;   /* 标题：楷书感 */
--font-serif:  "Source Han Serif SC", "Noto Serif SC", "STSong", "SimSun", serif;  /* 正文/原文：宋体衬线 */
--font-sans:   "Source Han Sans SC", "PingFang SC", system-ui, sans-serif;  /* UI 控件：黑体（克制） */
--font-mono:   "JetDM Mono", "SF Mono", monospace;  /* 数字/代码 */
```

- **原文区**：宋体衬线（`--font-serif`），18–22px，行高 1.8–2.0，字距微松——模拟版刻古籍的从容。
- **章节标题 / 书名**：楷体（`--font-title`），竖排题款感（见 §5）。
- **UI 按钮/导航**：黑体（`--font-sans`），小字号，不抢戏。
- **数字**：等宽（阅读进度、统计），用 tabular-nums。
- 可选 **webfont**：霞鹜文楷（开源楷体，GitHub 有）做标题/古文，气质对。CDN 或自托管子集。

## 4. 纹理与质感

这是把"扁平 UI"变成"有水墨气"的关键，但**点到为止**，不当背景图铺满。

| 质感 | 实现方式 | 用在哪 |
|---|---|---|
| **宣纸底纹** | CSS：`background-image` 叠一层极淡 SVG `feTurbulence` 噪点（opacity 0.03–0.05），模拟纸张纤维 | 全局 body 背景（仅浅色） |
| **水墨晕染** | 径向渐变 + `filter: blur()`，边缘虚化不规则 | 章节起首装饰、空状态插画、分隔处 |
| **飞白笔触** | SVG mask / CSS `mask-image`（枯笔笔刷），文字或分隔线带断续 | 大标题、水墨分隔线 |
| **印章质感** | 朱红方块 + 微微不规则边缘（`border-radius` 不均 + 噪点 mask）+ 篆体字 | logo、主按钮、落款 |

> 宣纸噪点用 SVG turbulence（性能好、可缩放），不要用位图纹理图（糊、重）。

## 5. 布局

| 手法 | 说明 | 应用 |
|---|---|---|
| **大量留白** | 内边距加大（阅读区左右各 8–12% 空白），卡片间 24–32px | 全局；阅读三栏尤其 |
| **不对称平衡** | 不强求左右等宽，主栏宽、辅栏窄，留白偏一侧 | 三栏比例 38/34/28，可调 |
| **竖排题款** | `writing-mode: vertical-rl`，章节标题/书名竖排小字 | 阅读页章节标题、书卡角标 |
| **卷轴/册页隐喻** | 章节切换如卷轴展开（横向）；段落如册页 | 阅读翻章动效；书库卡片册页式 |
| **题跋式标注** | 解读/批注以小字 + 朱印落款，仿古籍批注 | 解读栏、笔记引用 |

## 6. 组件设计

### 6.1 按钮
- **主按钮**：印章式——朱砂底（`cinnabar`）+ 白/米字 + 微不规则圆角 + 可选篆体。hover 加深。
- **次按钮**：墨块——宣纸底 + 细墨边 + 墨字。或纯文字 + 下划线（墨痕）。
- **图标按钮**：线性（铁线描），1.5px 描边，方头，无填充。**禁用 emoji 和实心圆角填充图标**。

### 6.2 卡片（书卡 / 笔记卡）
- 宣纸底 + 1px 淡墨边（`border: 1px solid rgba(28,26,23,0.1)`）+ **无阴影**（靠留白和边线分层，不靠投影）。
- 左上角可加朱印小标（书名首字 / 进度）。
- hover：边线变浓墨，不抬升。

### 6.3 分隔线
- **水墨笔触线**：SVG path（粗细变化 + 飞白），用于章节/段落大分隔。
- **细铁线**：1px 淡墨直线，用于列表项。
- 禁用渐变分隔、粗实线。

### 6.4 输入 / 搜索框
- 宣纸底 + 底部一道墨痕（只底线，无边框盒子）——仿毛笔格线。
- focus 时底线变朱砂。

### 6.5 进度 / 滑块
- 阅读进度：墨痕条（深浅渐变，模拟墨色由淡到浓）。
- 记忆卡掌握度：墨色环形（浓 = 已掌握）。

### 6.6 标签
- 小印玺式：淡墨底 + 墨字 + 方角；或朱砂描边。

## 7. 动效（克制）

| 动效 | 触发 | 实现 |
|---|---|---|
| **墨晕渐显** | 页面/内容进入 | `opacity` + 轻微 `blur(4px)→0`，300ms ease-out |
| **卷轴展开** | 章节切换 | 宽度/位移 + 内容淡入（横向卷轴感） |
| **描红书写** | 大标题首次出现（可选） | SVG stroke-dashoffset 动画（笔画顺序写出）——慎用，仅 hero |
| **印章按下** | 主按钮点击 | scale 0.96 + 朱砂微闪 |
| **翻卡** | 记忆卡翻转 | 现有翻转 + 墨色过渡 |

> 全部 200–400ms，ease-out。**禁弹跳、禁长动画、禁自动播放装饰**——阅读类工具，动效服务专注。

## 8. 参考与灵感（调研来源）

- **宋代美学**：汝窑天青（克制单色）、宋版书（版刻字体、鱼尾版口、左右双栏）、宋人小品画（大量留白）——**最高参考**，克制典雅。
- **古书籍形制**：线装书、卷轴、册页、经折装；题跋、批注、圈点、朱印。
- **水墨画**：写意（八大、齐白石、徐渭）——笔墨简、留白多；工笔（宋画）——细线、淡彩。
- **传统色**：《中国传统色》（故宫色）：牙色、月白、朱砂、赭石、花青、黛。
- **新中式 UI 参考**：故宫文创官网、观夏（香薰品牌站，留白与宋韵）、某些古风阅读 app（取其留白与衬线，弃其过度纹饰）。
- **反例（避坑）**：满屏祥云底纹、毛笔字标题滥用、大红大金、龙凤纹——"旅游纪念品风"，非高级国风。

## 9. 应用到本项目的各模块

| 模块 | 水墨风适配 |
|---|---|
| **阅读三栏（核心）** | 原文栏：宋体衬线大字 + 行高 1.9 + 左右留白；章节标题竖排楷书题款；解读栏淡绛底（虚）；资源栏水墨配图淡彩。逐段同步滚动加墨晕过渡。 |
| **书库** | 书卡 = 册页（宣纸底 + 细墨边 + 朱印书名）；导入按钮 = 印章式；空状态 = 一笔水墨留白插画。 |
| **检索** | 搜索框底线墨痕；结果 snippet 高亮用朱砂（非黄）；命中列表册页式。 |
| **记忆卡** | 卡片正反面宣纸质感；翻卡墨色过渡；仪表盘热力图用墨色阶（浓 = 高频）。 |
| **笔记** | 编辑区仿稿纸（淡格线）；双链 `[[ ]]` 朱砂下划；导出 PDF 自带宣纸底 + 落款。 |
| **设置** | 极简表单，底线输入；AI 状态用小印玺。 |
| **全局** | 主题 token 重构（§2 色彩 + §3 字体）；body 宣纸噪点；导航栏墨痕底；logo 印章。 |

## 10. 实施路线（分阶段，可逐步落地）

1. **Phase A · 主题 token 重构**（地基，影响全局）：替换 `src/styles/theme.css` 的 paper/ink/dark 三套 token 为 §2 色彩 + §3 字体变量；接入霞鹜文楷（标题）。**改动小、收益最大**。
2. **Phase B · 宣纸纹理 + 墨痕**：body 叠 SVG 噪点；分隔线换水墨笔触；输入框底线化。
3. **Phase C · 组件换装**：书卡册页化、主按钮印章化、标签印玺化、进度墨色。
4. **Phase D · 阅读区精修**：原文竖排标题/衬线大字/留白；解读淡彩；翻章卷轴动效。
5. **Phase E · 装饰收尾**：空状态水墨插画、角落晕染、落款印章、描红标题（hero）。

> 每阶段独立可交付，A 完成即有明显气质提升。建议从 A 开始，看效果再决定 B–E 深度。

## 11. 待确认（实施前需定）

1. **楷书 webfont**：用霞鹜文楷（开源，~自托管子集）还是系统楷体（零体积但各机渲染不一）？建议文楷做标题/古文。
2. **深色模式优先级**：先做宣纸浅色（主），夜墨深色同步还是后续？
3. **装饰程度**：走"极简宋韵"（几乎纯黑白 + 朱砂，最克制）还是"水墨写意"（加晕染/笔触装饰，更有画意）？建议前者打底、后者在阅读区点缀。
4. **竖排**：原文区是否竖排？竖排古意浓但长段落阅读效率降——建议**仅章节标题/书名竖排**，正文横排。
