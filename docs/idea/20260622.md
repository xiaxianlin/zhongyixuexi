# 详情页功能调整

## 功能改造
- 书籍进行分类，分为古籍和现代书籍
- 章节不再分段，直接展示章节全部内容，AI 分析功能根据章节进行分析，包括解读、医理，如果是古籍的话需要翻译成白话
- 章节改成树形结构对应多章节多小节
- 正文内容可以进行部分选中，选中后可以进行摘录、笔记、AI 分析

## UI 改造
- 章模块改成树选择器，并可以对章节和小节进行增删改
- 段模块改成阅读区，可以用光标选中文字，选中弹出弹出三个按钮分别为摘录、写笔记、引用，引用是将这段话引用到 AI 对话框里面
- 析模块改造成多功能区域，是一个 tab 选项卡，主要包括对话、解读、医理、白话、笔记、摘录，默认展示 AI 对话框，选项卡放在右边，文字竖排
- AI 分析按钮放到阅读区右上角，章节内容可以被修改
